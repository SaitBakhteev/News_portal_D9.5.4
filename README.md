Склонируйте проект.
Установите и активируйте виртуальное окружение.
Закройте проект, удалите папку .idea и вновь перезапустите проект, чтобы инициализировались новые конфигурации соединения виртуального окружения с проектом.
Установите зависимости из файла requirements.
Если в процессе установки библиотек вознкнет ошибка, то установите зависимости командой "python.exe -m pip install -r .\requirements"
Создайте .env файл в папке djangoProject_News_Portal

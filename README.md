1. Склонируйте проект.
2. Установите и активируйте виртуальное окружение.
3. Закройте проект, удалите папку .idea и вновь перезапустите проект, чтобы инициализировались новые конфигурации соединения виртуального окружения с проектом.
4. Установите зависимости из файла requirements.
5. Если в процессе установки библиотек вознкнет ошибка, то установите зависимости командой "python.exe -m pip install -r .\requirements"
6. Создайте .env файл в папке djangoProject_News_Portal

Содержимое .env файла должно быть следующим:
`SECRET_KEY=####<Ваш секретный код django>####

>данные для почты
EMAIL_HOST_USER=####<Ваш хост почты>####
DEFAULT_FROM_EMAIL=####<Ваша адрес подключенной для рассылки почты>####
EMAIL_HOST_PASSWORD=####<специальный пароль почты для рассылки>####

>данные для oauth yandex
YANDEX_CLIEND_ID=####<Ваш уникальный id, выданный **yandex**####
YANDEX_SECRET=####<Ваш секретный пароль, выданный **yandex**####`
